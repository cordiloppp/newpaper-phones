# Newspaper Phones

## Descripción

Newspaper Phones es un sitio web de noticias dedicado a la tecnología móvil. Proporciona reseñas, actualizaciones y análisis de los últimos smartphones de marcas líderes como Samsung, Huawei, Xiaomi y Apple. El sitio está diseñado con un enfoque en la experiencia del usuario, utilizando HTML, CSS y JavaScript para una navegación fluida y responsiva.

## Características

- **Noticias Actualizadas**: Artículos sobre los últimos lanzamientos y tendencias en smartphones.
- **Reseñas de Productos**: Análisis detallados de modelos específicos con especificaciones técnicas.
- **Diseño Responsivo**: Optimizado para dispositivos móviles, tablets y desktops.
- **Interfaz Moderna**: Utiliza CSS Grid y Flexbox para un layout atractivo.
- **Animaciones**: Integración de ScrollReveal para efectos de scroll suaves.

## Tecnologías Utilizadas

- **HTML5**: Estructura del sitio web.
- **CSS3**: Estilos y diseño responsivo, incluyendo media queries.
- **JavaScript**: Interactividad básica y animaciones con ScrollReveal.
- **Font Awesome**: Iconos para navegación y elementos visuales.
- **Google Fonts**: Tipografía Oswald para un look moderno.

## Instalación y Uso

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/tu-usuario/newpaper-phones.git
   cd newpaper-phones
   ```

2. **Abre el proyecto**:
   - Abre `index.html` en tu navegador web preferido.
   - Para un servidor local, puedes usar una extensión como "Live Server" en VS Code o cualquier servidor HTTP simple.

3. **Requisitos**:
   - Navegador web moderno (Chrome, Firefox, Safari, Edge).
   - Conexión a internet para cargar fuentes externas (Google Fonts, Font Awesome).
